# MVP App de Intermediación Laboral para Vendedores Ambulantes de TransMilenio

Aplicación móvil desarrollada en **Flutter** como parte de un trabajo de grado de **Ingeniería de Sistemas y Computación** de la **Universidad Católica de Colombia**.

## Descripción

Este proyecto propone el diseño y desarrollo de un **MVP (Producto Mínimo Viable)** de una aplicación móvil orientada a facilitar la **inserción laboral formal** de vendedores ambulantes del sistema **TransMilenio de Bogotá**, conectándolos con oportunidades de **empleo** y **formación** mediante una solución digital accesible, comprensible e inclusiva.

La iniciativa surge como respuesta a problemáticas asociadas a la informalidad laboral, las barreras de acceso al empleo formal, la baja disponibilidad de canales de intermediación laboral para esta población y la necesidad de fortalecer rutas de inclusión productiva.

## Objetivo general

Diseñar un MVP de aplicación móvil que conecte a los vendedores informales del sistema TransMilenio con oportunidades laborales y de formación, mediante un entorno accesible e inclusivo, favoreciendo su inserción en el mercado laboral formal.

## Objetivos específicos

- Caracterizar necesidades, barreras y aspiraciones laborales de la población objetivo.
- Identificar requerimientos funcionales y no funcionales de la aplicación.
- Desarrollar un MVP navegable con criterios de usabilidad, accesibilidad y seguridad.
- Evaluar la usabilidad y funcionalidad del prototipo con usuarios.

## Contexto del proyecto

En Bogotá, la informalidad laboral representa una barrera importante para el acceso a empleo digno, estabilidad económica y protección social. Dentro de este panorama, los vendedores ambulantes que operan en el sistema TransMilenio enfrentan dificultades relacionadas con:

- acceso limitado a ofertas laborales formales,
- barreras de formación y documentación,
- baja alfabetización digital en algunos casos,
- estigmatización social,
- condiciones laborales precarias.

Frente a esto, la aplicación busca servir como un canal digital de apoyo para acercar esta población a oportunidades de empleabilidad y formación, aportando al mismo tiempo a procesos de inclusión social y productiva.

## Alcance del MVP

Este repositorio contiene el desarrollo del MVP de la aplicación móvil. El alcance del proyecto está orientado a validar una solución funcional inicial, por lo que puede incluir módulos como:

- onboarding o bienvenida,
- registro e identificación básica del usuario,
- perfilamiento laboral,
- visualización de ofertas laborales,
- acceso a oportunidades de formación,
- postulación a vacantes,
- navegación simple y comprensible,
- persistencia local de datos para pruebas del prototipo.

> Nota: al tratarse de un MVP académico, algunas funcionalidades pueden estar simuladas o implementadas de manera parcial para efectos de validación.

## Stack tecnológico

- **Flutter**: framework principal para el desarrollo móvil multiplataforma.
- **Dart**: lenguaje de programación.
- **SQLite**: base de datos local para persistencia en el MVP.
- **GitHub**: control de versiones y gestión del repositorio.
- **Figma**: apoyo para wireframes, flujo de navegación y diseño UI/UX.
